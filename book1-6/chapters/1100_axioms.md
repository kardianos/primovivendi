%+ **Reality is real, knowable, and consequential.** There is a world which exists whether any man likes it, names it, or votes upon it; the world can be known, not perfectly but with usable confidence; and its facts carry consequences which no declaration can set aside. Truth is correspondence to this reality: a claim is true when the claim matches what is. %- **Justice is procedural symmetry.** Within an honest scope, the same rules and methods of judgment apply to each person in the same position. Two rivals answer otherwise. Outcome symmetry holds that justice is the same end-state for everyone and treats disparity as proof of injustice. Value symmetry holds that justice is equal return in kind, a ledger in which injury answers injury. The chain runs: limited, self-interested people must cooperate; cooperation needs stable expectations; stable expectations need the same rules within a defined set of people and roles. Outcome symmetry must apply unequal procedure to reach its equal end-states, and the work of equalizing breeds a class of officials who live by it. Value symmetry faces backward and has no end, because each injury answered becomes the next injury to answer. The same form holds across domains: in law, one rule for the same case; in inquiry, one critical standard for every model; in conduct, the reversal test (would you still accept the rule if the positions were swapped); between generations, the procedure you wish had been used toward you (life given, formation given, a platform to stand on).
